Auto-update timer

Ak si vo wizarde povolil auto-update, vytvorí sa systemd timer (netbox-update.timer), ktorý raz týždenne spustí docker compose pull && up -d a prune. Cesta je idempotentná — recreatne len zmenené kontajnery, dáta v named volumes ostávajú netknuté.

Produkcia

Pre kontrolovaný upgrade pinuj konkrétny tag a auto-update vypni — verziu meň vedome a otestovane, nie automaticky na to, čo práve vyšlo.
