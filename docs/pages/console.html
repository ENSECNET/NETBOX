Appliance console

Status obrazovka a správcovské menu, ktoré sa automaticky spustí pri prihlásení do kontajnera (pct enter 310). Štýl pfSense / OPNsense / Home Assistant OS.

Ako vyzerá

   6) Shell (root bash)
   7) Reboot appliance
   0) Logout

Stavové body

Body pri službách sú farebné a živé: zelený ● running, žltý ● starting, červený ● stopped. Stav sa číta cez docker ps pri každom zobrazení.

Inštalácia

Konzolu nainštaluje deployer do /usr/local/sbin/netbox-console.sh a cez /etc/profile.d/ ju spustí pri interaktívnom logine. Žiadne príkazy na zapamätanie.
