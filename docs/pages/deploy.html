Deployment walk-through

Celý deploy krok za krokom s reálnym výstupom. Sieťové hodnoty sú RFC 5737 placeholdery (192.0.2.0/24).

1 · Jeden príkaz na Proxmox node-e

V shelli node-u (web UI → node → Shell, alebo SSH):

bash -c "$(curl -fsSL https://raw.githubusercontent.com/ensecnet/netbox-lxc/main/scripts/netbox-lxc-deploy.sh)"
Radšej najprv prečítať?

git clone → less skript → spustiť. Skript je krátky a čitateľný.

2 · Wizard

Pýta sa na všetko vopred, defaulty v zátvorkách. Sekvencia: VMID, hostname, root heslo, NetBox admin (user/email/heslo), výpis storage + voľba, disk/RAM/CPU, HTTP port, potom sieť (bridge/IP/GW/DNS/VLAN) a verzie (netbox-docker tag, auto-update). Na konci súhrn a jedno potvrdenie.
